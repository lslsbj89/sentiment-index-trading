## 🚀 优化建议

### 建议1: 加入动态止损

```python
# 当前: 只有超时卖出
if holding_days >= 60:
    SELL

# 优化: 加入止损
if holding_days >= 60:
    SELL
elif profit_pct <= -15%:  # 动态止损
    SELL
```

**预期效果**: 避免单笔亏损>15%，减少2022年熊市损失

---

### 建议2: 根据指数动态调整持仓

```python
# 当前: 固定60天
holding_days = 60

# 优化: 根据指数动态调整
if current_idx < -20:  # 极度恐慌
    holding_days = 90  # 延长持仓，等待反弹
elif current_idx > 50:  # 极度贪婪
    holding_days = 30  # 缩短持仓，及时止盈
else:
    holding_days = 60  # 正常持仓
```

**预期效果**: 熊市减少亏损，牛市及时止盈

---

### 建议3: 分批建仓

```python
# 当前: 一次性买入80%仓位
shares = cash * 0.8 / price

# 优化: 分批买入
if current_idx in (0, 10):  # 轻度恐慌
    position_pct = 0.4  # 先买40%
elif current_idx < 0:  # 深度恐慌
    position_pct = 0.6  # 买60%
elif current_idx < -10:  # 极度恐慌
    position_pct = 0.8  # 买80%
```

**预期效果**: 越跌越买，降低平均成本
