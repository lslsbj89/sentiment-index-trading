## 🛠️ 优化建议

### 1. 加入止损

```python
if profit_pct <= -15%:
    SELL  # 止损
```

### 2. 动态持仓

```python
if idx < -20:
    max_holding = 90  # 熊市延长
elif idx > 50:
    max_holding = 30  # 牛市缩短
else:
    max_holding = 60
```

### 3. 分批建仓

```python
if idx < -10:
    position = 0.8  # 极度恐慌→重仓
elif idx < 0:
    position = 0.6
else:
    position = 0.4  # 轻度恐慌→轻仓
```
