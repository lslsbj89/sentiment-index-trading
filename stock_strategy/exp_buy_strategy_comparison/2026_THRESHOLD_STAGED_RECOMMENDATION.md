## 二、核心策略

### 2.1 阈值分批买入规则

**买入逻辑** (统一适用所有股票):
```
当 Sentiment < 5  → 买入第1档 (25%)
当 Sentiment < 0  → 买入第2档 (25%)
当 Sentiment < -5 → 买入第3档 (25%)
当 Sentiment < -10 → 买入第4档 (25%)
```

**特点**:
- 情绪越恐慌，买入越多
- 自动摊低成本
- 最多投入100%配置资金

### 2.2 卖出规则 (各股票独立参数)

**卖出逻辑** (满足任一即卖出):
```
1. OR条件: Sentiment > OR阈值 → 无条件卖出
2. AND条件: Sentiment > AND阈值 且 Price < MA50 → 卖出
```

### 2.3 分批基准重置

```
每次清仓卖出后:
  新分批基准 = 当前总资产
  每档金额 = 新分批基准 × 25%
```

这实现了**动态复利**效果。

## 六、操作流程

### 6.1 每日检查清单

```
□ 1. 获取当日各股票 Fear Greed Index (S=3)
□ 2. 检查各股票情绪是否触发买入档位
     - sent < 5  → Batch1 (若未买)
     - sent < 0  → Batch2 (若未买)
     - sent < -5 → Batch3 (若未买)
     - sent < -10 → Batch4 (若未买)
□ 3. 检查持仓是否触发卖出信号
     - OR条件: sent > OR阈值
     - AND条件: sent > AND阈值 且 price < MA50
□ 4. 检查止损条件 (亏损超过5-7%)
□ 5. 记录交易日志
```

### 6.2 卖出后重置

```
卖出清仓后:
□ 1. 计算新总资产
□ 2. 更新分批基准 = 新总资产 × 配置比例
□ 3. 重置买入档位状态 (全部清零)
□ 4. 等待新的买入信号
```
