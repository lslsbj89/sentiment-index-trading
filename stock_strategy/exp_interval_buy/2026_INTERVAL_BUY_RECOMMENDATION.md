# 2026年美股七强操作建议 - 间隔分批策略

**生成日期**: 2026-01-28
**策略版本**: v3.0 (间隔分批买入)
**基于**: Walk-Forward 滚动验证 (W2020-W2025, 6个窗口)
**数据源**: Fear Greed Index S3 (Smoothing=3, MoneyFlow=13)
**数据表**: `fear_greed_index_s3`

## 一、核心结论

1. **间隔分批买入优于阈值分批**: 平均收益 +411% vs +324%，提升 +87%
2. **资金利用率大幅提升**: 从 25-100% 提升至 ~100%
3. **买入阈值 buy<0 平均最优**: +431.5%，4/7 股票最优；buy<5 次优 +410.9%
4. **各股票最优阈值不同**: NVDA/MSFT/AMZN/META→buy<0, AAPL→buy<5, GOOGL→buy<-5, TSLA→buy<-15
5. **卖出参数由 Walk-Forward 训练决定**: 每年重新优化，不固定
6. **固定保守参数优于联合优化**: 联合优化过拟合，固定参数更稳健

## 三、买入策略 (间隔分批买入)

### 3.1 参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| buy_threshold | **0** (统一) | 情绪指数 < 0 触发买入模式，平均最优 |
| interval_days | 7 | 每隔 7 天买入一批 |
| batch_pct | 20% | 每批买入基础资金的 20% |

**统一阈值 vs 个股阈值**:

| 方案 | 阈值 | 平均收益 | 适用场景 |
|------|------|---------|---------|
| **方案A: 统一 buy<0** | 全部 <0 | +431.5% | 简单执行，推荐 |
| 方案B: 统一 buy<5 | 全部 <5 | +410.9% | 保守备选 |
| 方案C: 个股最优 | 见下表 | 各股最优 | 精细化管理 |

**方案C 个股最优阈值**:

| 股票 | 最优阈值 | 最优收益 | 说明 |
|------|---------|---------|------|
| NVDA | buy<0 | +1510.7% | 比 buy<5 高 +60% |
| TSLA | buy<-15 | +517.1% | 极端恐慌时抄底最优 |
| AAPL | buy<5 | +302.9% | 快速入场最优 |
| MSFT | buy<0 | +219.8% | 比 buy<5 高 +37% |
| GOOGL | buy<-5 | +282.1% | 适度等待最优 |
| AMZN | buy<0 | +171.4% | 比 buy<5 高 +29% |
| META | buy<0 | +160.5% | 比 buy<5 高 +51% |

### 3.2 执行流程

```
第1天: 情绪指数 < 0 (推荐阈值)
  → 进入买入模式
  → 立即买入首批 (配置资金的 20%)
  → 记录买入日期

第8天: 仍在买入模式中 (距上次买入 >= 7天)
  → 买入第2批 (配置资金的 20%)

第15天: 仍在买入模式中
  → 买入第3批 (配置资金的 20%)

...

某天: 情绪指数回升 >= 0
  → 一次性买入剩余全部资金
  → 退出买入模式
```

### 3.3 操作示例 (以 NVDA $25,000, buy<0 为例)

| 阶段 | 情绪 | 操作 | 金额 | 累计投入 |
|------|------|------|------|---------|
| Day 1 | -0.5 < 0 | 首批买入 | $5,000 (20%) | $5,000 |
| Day 8 | -3.2 < 0 | 间隔买入 | $5,000 (20%) | $10,000 |
| Day 14 | 1.8 >= 0 | 恢复全买 | $15,000 (剩余) | $25,000 |

### 3.4 关键规则

- **一只股票同时只有一个买入模式**: 买入模式中不会重复进入
- **恢复全买是核心**: 情绪回升至阈值时一次性投入剩余资金，确保 ~100% 资金利用
- **卖出清仓后重置**: 基准资金更新为当前总资产 × 配置比例（动态复利）
- **跨年连续**: 买入模式在年度窗口间传递，不因换年重置
- **阈值选择**: 统一用 buy<0 最简单；追求极致可按个股最优阈值执行

## 四、卖出策略

### 4.1 双条件卖出机制

| 条件 | 触发规则 | 优先级 | 说明 |
|------|---------|--------|------|
| **OR 条件** | sentiment > OR阈值 | 高 | 情绪极度贪婪，无条件卖出 |
| **AND 条件** | sentiment > AND阈值 **且** price < MA50 | 中 | 情绪偏高 + 价格转弱 |

### 4.2 2026年卖出参数 (W2026, 训练期 2022-2025)

通过网格搜索 AND×OR = 25 组合，在训练期 (2022-2025) 用原版阈值分批买入回测，选训练收益最高的参数:

| 股票 | AND阈值 | OR阈值 | 训练收益 | 卖出特征 |
|------|---------|--------|---------|---------|
| **NVDA** | >5 | >50 | +381.4% | 敏感型: 情绪稍高+价跌MA50即卖 |
| **TSLA** | >25 | >60 | +245.9% | 宽松型: 允许长时间持有 |
| **AAPL** | >5 | >30 | +128.9% | 最敏感: 快进快出 |
| **MSFT** | >15 | >30 | +59.7% | 稳定型: 7个窗口参数完全一致 |
| **GOOGL** | >15 | >50 | +127.3% | 中等偏松: OR从历史30升至50 |
| **AMZN** | >10 | >30 | +64.4% | 中等敏感 |
| **META** | >15 | >55 | +121.9% | 稳定型: 近3窗口参数一致 |

### 4.3 历史窗口卖出参数演变

| 股票 | W2020 | W2021 | W2022 | W2023 | W2024 | W2025 | **W2026** |
|------|-------|-------|-------|-------|-------|-------|-----------|

**W2026 参数变化分析**:
- NVDA AND 从 25→5: 训练期包含 2022 熊市，模型倾向更频繁止盈
- AAPL AND 从 10→5: 同上，趋向快进快出
- GOOGL OR 从 30→50: 训练期 2023-2025 反弹，模型学会持有更久
- MSFT/META: 参数完全不变，最稳定

### 4.3 卖出后重置

```
卖出清仓后:
  1. 计算当前总资产
  2. 更新各股票配置资金 = 总资产 × 配置比例
  3. 重置买入模式状态 (退出买入模式)
  4. 等待下一次情绪 < 0 (或个股阈值) 触发新一轮买入
```

## 七、每日操作流程

### 7.1 检查清单

```
□ 1. 获取各股票当日 Fear Greed Index (S3, MF=13)
     数据表: fear_greed_index_s3

□ 2. 检查是否触发卖出
     对每只持仓股票:
       - OR: 情绪 > OR阈值 → 无条件卖出全部
       - AND: 情绪 > AND阈值 且 价格 < MA50 → 卖出全部
     卖出后: 重置买入模式，更新配置基准

□ 3. 检查是否触发买入
     对每只无持仓/在买入模式中的股票:
       - 无持仓 且 情绪 < 0 (或个股阈值) → 进入买入模式，首批买入 20%
       - 买入模式中 且 距上次买入 >= 7天 → 再买 20%
       - 买入模式中 且 情绪 >= 0 (或个股阈值) → 一次性买入全部剩余

□ 4. 记录交易日志
```

## 八、风险控制

### 8.1 策略特性

| 市场环境 | 策略表现 | 建议 |
|---------|---------|------|
| 震荡市 | 最佳 (2025年: 5/7胜率) | 严格执行 |
| 熊市 | 良好 (分批抄底摊低成本) | 严格执行，坚持等待 |
| 牛市 | 一般 (卖出过早，跑输持有) | 可适当提高 OR 阈值 |

### 8.2 熊市风险 (2022年教训)

| 股票 | 2022年 | 说明 |
|------|--------|------|
| TSLA | -53.7% | 分批买入后继续暴跌 |
| META | -51.9% | 被套无卖出信号 |
| NVDA | -39.5% | 分批全买后回调 |
| AMZN | -39.0% | 长期套牢 |

**应对**:
- 间隔分批比阈值分批风险分散更好 (时间维度分散)
- 熊市期间保持耐心，不追加超配
- 可将现金储备从 10% 提高至 20%

### 8.3 过拟合风险

- 联合优化 (买入+卖出参数同时搜索) 导致严重过拟合: NVDA +513% vs 固定参数 +1451%
- **原则**: 买入参数固定 (buy<0, 7天, 20%)，仅优化卖出参数
- 每年使用 Walk-Forward 重新训练卖出参数
