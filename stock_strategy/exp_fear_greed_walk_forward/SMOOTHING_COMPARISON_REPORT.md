## 一、平滑参数说明

### RMA 公式

```
RMA_t = α × Price_t + (1-α) × RMA_{t-1}
其中 α = 1/n
```

| 参数 | α值 | 当日权重 | 特点 |
|------|-----|---------|------|
| Smoothing=3 | 1/3 = 0.333 | 33.3% | 响应快，波动大 |
| Smoothing=5 | 1/5 = 0.200 | 20.0% | 响应慢，更平滑 |

## 七、结论

### 7.1 核心发现

1. **平滑参数显著影响策略表现**
   - 平均收益差异: +43.4% (Smooth=3更高)
   - 但窗口胜率: Smooth=5更高 (22 vs 11)

2. **参数选择应因股票而异**
   - 高波动股票 → Smooth=3 (NVDA +311%, TSLA +136%)
   - 稳定股票 → Smooth=5 (AMZN +58%, GOOGL +37%)

3. **风险收益权衡**
   - Smooth=3: 高收益、高波动、更多交易
   - Smooth=5: 稳定收益、低波动、较少交易

### 7.2 实操建议

| 建议 | 说明 |
|------|------|
| 分组配置 | 按股票波动性选择不同平滑参数 |
| 动态调整 | 牛市用S=3，熊市用S=5 |
| 风险控制 | S=3组设更严格止损 (-10%) |
| 定期复盘 | 每季度对比两组表现 |
