## 买入策略 (固定参数, 不需要训练)

**当前方案**: 统一阈值 buy<0 (平均收益最高 +431.5%, 4/7股票最优)

| 参数 | 值 |
|------|------|
| 买入触发 | 情绪 < 0 |
| 间隔天数 | 7 天 |
| 每批比例 | 20% |

**流程**:
1. 情绪 < 0 → 进入买入模式, 买入配置资金的 20%
2. 每隔 7 天 → 再买 20%
3. 情绪回升 >= 0 → 买入全部剩余, 退出买入模式

**卖出清仓后**: 基准资金 = 当前总资产 × 配置比例 (动态复利)

### 个股最优买入阈值 (参考)

| 股票 | 最优阈值 | 最优收益 | 与统一buy<0差异 |
|------|---------|---------|---------------|
| NVDA | buy<0 | +1510.7% | 一致 |
| TSLA | buy<-15 | +517.1% | 极端恐慌抄底 |
| AAPL | buy<5 | +302.9% | 更早入场 |
| MSFT | buy<0 | +219.8% | 一致 |
| GOOGL | buy<-5 | +282.1% | 适度等待 |
| AMZN | buy<0 | +171.4% | 一致 |
| META | buy<0 | +160.5% | 一致 |

> 个股最优阈值来自 Walk-Forward 回测 (7窗口×5阈值)。统一 buy<0 已是全局最优, 个股阈值仅供参考, 存在过拟合风险。

## 卖出策略

**双条件卖出** (满足任一即全部卖出):
- **OR**: 情绪 > OR阈值 → 无条件卖出
- **AND**: 情绪 > AND阈值 **且** 价格 < MA50 → 卖出

### 2026-Q1 卖出参数 (训练期 2022-2025, interval buy training)

| 股票 | AND | OR | 训练收益 | 卖出含义 |
|------|-----|-----|---------|---------|
| NVDA | >5 | >50 | +517.9% | 敏感, 情绪稍高+价跌MA50即卖 |
| TSLA | >25 | >30 | +233.6% | AND宽松, OR敏感 |
| AAPL | >5 | >30 | +111.4% | 最敏感, 快进快出 |
| MSFT | >5 | >30 | +75.0% | 敏感, 快进快出 |
| GOOGL | >10 | >50 | +137.4% | 中等偏松 |
| AMZN | >10 | >30 | +88.3% | 中等 |
| META | >15 | >55 | +141.9% | 稳定 |

> **训练方法**: 使用 interval buy training (训练期买入策略与生产测试一致: 情绪<0, 每7天20%, 回升全买)。
> 相比旧版 staged buy training, 3只股票参数发生变化: TSLA OR 60→30, MSFT AND 15→5, GOOGL AND 15→10。

## 每日检查

```
1. 获取各股票情绪指数 (fear_greed_index_s3)

2. 持仓股 → 检查卖出
   情绪 > OR阈值?          → 卖出全部
   情绪 > AND阈值 且 < MA50? → 卖出全部
   卖出后重置买入状态, 更新配置基准

3. 空仓/买入模式中 → 检查买入
   空仓 且 情绪 < 0?        → 进入买入模式, 买 20%
   买入模式 且 >=7天?        → 再买 20%
   买入模式 且 情绪 >= 0?    → 买入全部剩余
```

